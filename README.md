# chat-bot

Bot for telegram

## Local up

1. install python or setup virtual env
2. requirements install

   ```pip install -r requirements.txt```

### Start

run `python main.py`


#### command:
1) /start   
  Hello, {user first name}!!!'

2) /duck
  generate random img with duck

3) hangman
  start game

4) /speech
generate your text to speech

5) /voice
generate your speech to text
